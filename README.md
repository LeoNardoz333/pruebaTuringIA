Instrucciones de configuración:
Ejecutar el script de la base de datos ubicado en la carpeta de "Base de Datos" para crear el usuario y la base de datos del proyecto.

Ejecutar en una terminal dentro de la carpeta prueba-TuringIA:

php artisan migrate

Para crear las tablas de la base de datos

Ejecutar en una terminal dentro de la misma carpeta los comandos:

php artisan serve
npm run dev

Para poder inicar el proyecto, acceder al vínculo: http://127.0.0.1:8000/ desde un navegador.
